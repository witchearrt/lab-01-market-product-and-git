## Roles and responsibilities

1. **Back-end engineer** — develops server-side logic, APIs, and microservices. Responsible for data processing, database integration, and ensuring scalability and performance of server components.

2. **Mobile engineer (iOS/Android)** — builds and maintains mobile applications for iOS and Android platforms. Implements user interfaces, server API integration, local storage handling, and native device features.

3. **DevOps Engineer** — automates deployment, monitoring, and infrastructure scaling processes. Manages CI/CD pipelines, containerization, orchestration, and ensures production environment reliability.

4. **Security Engineer** — designs and implements security measures to protect user data. Conducts code audits, vulnerability testing, manages encryption, and ensures compliance with security standards.

5. **QA Engineer** — develops and executes tests to verify product functionality, performance, and security. Automates testing, identifies defects, and ensures release quality.

## Common skills across roles

- **Programming languages** — proficiency in relevant languages (Python, Go, Java, Swift, Kotlin, JavaScript/TypeScript) depending on role and platform.
- **Version control (Git)** — ability to work with version control systems for collaborative development and code reviews.
- **API design** — understanding of REST/gRPC API design principles for component communication.
- **Testing** — skills in writing unit, integration, and end-to-end tests to ensure code quality.
- **Debugging and monitoring** — ability to use debugging, logging, and monitoring tools to identify and resolve issues.
- **Cloud platforms** — basic understanding of cloud infrastructures (AWS, GCP, Azure) and distributed systems principles.
